# Domain: rendering

## Purpose
Capture practical rendering strategies used for lunar and solar eclipse visuals within the Moon Simulator interface.

## Scope
Included in this domain:
- EclipseObserver UI setup and render loop behavior
- Moon mesh creation and texture handling workflows
- Color space and layered disc techniques for solar/lunar transitions

Excluded from this domain:
- Orbital mechanics simulation logic
- Localization content outside eclipse observer labels

## Ownership
Graphics Team

## Usage
Refer to this domain when adjusting eclipse visuals, color blending, and shader behavior in the observer UI.
