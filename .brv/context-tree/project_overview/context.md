# Domain: project_overview

## Purpose
Capture the high-level project overview, architecture patterns, and contribution guardrails for the Moon Phases Simulator.

## Scope
Included in this domain:
- Project stack, structure, and build/runtime commands
- Architecture patterns for scene, simulation, and UI layering
- Coding/contribution rules and persistent instructions for contributors

Excluded from this domain:
- Implementation details for specific features (e.g., individual components or shaders)
- Runtime data collected during simulations

## Ownership
Context engineering team

## Usage
Refer to this domain when documenting project-wide expectations, architecture, and onboarding guidance.
